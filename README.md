# Junior Developer

Organize What I Studied until I Became a Junior Developer<br>

주니어 개발자가 될 때까지 공부한 것을 정리한다<br>

머리로 이해한 것보다 기억하는 속도를 따라가지 못한다. 이제 더 이상 내 머리를 너무 믿지 말자.<br>

기록한 것은 매일 매일 읽어보고 내가 공부한 지식들을 잊지 않도록 하기!<br>
